# GazeEstimation


This is a Gaze Detector 

## Build Linux
    git clone --recursive https://github.com/apletea/GazeEstimation
    cd GazeEstimation
    mkdir build
    cd build
    cmake ..
    make -j6
    cd bin
    ./demo


Here some options but most common used input_vid:

    --input_vid=$path_to_video
